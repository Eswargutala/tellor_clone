# 📋 Trello Clone (Kanban Board)

A simple **Trello‑style web application** where users can add and organize tasks into columns like *Today, Week, Later*.  
This project demonstrates DOM manipulation, drag‑drop UI, and interactive task cards using **HTML, CSS & JavaScript**.

👉 **Live Demo:** https://eswargutala.github.io/tellor_clone/

---

## 📝 Features
- Add new tasks to any column
- Drag and drop tasks between columns
- Clean Kanban‑style interface
- Works on desktop and mobile browsers

---

## 🛠️ Technologies Used
- **HTML** – Page structure
- **CSS** – Layout and styling
- **JavaScript** – Task logic and interactivity

---

## 🚀 How to Use
1. Open your browser.
2. Go to the **Live Demo** link above.
3. Click **Add Task** under any column.
4. Enter the task and press **Enter** (or click add).
5. Drag and drop the task cards between columns as needed.
